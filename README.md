# authembed

## Description
authembed is a cool project built with <a href="http://nodejs.org/">Node.js</a> and <a href="https://github.com/nestjs/nest">nest.js</a> framework.

## Installation


```bash
$ yarn
```



## Running the app



```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```



## Test



```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
